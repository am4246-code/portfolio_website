# Amiyah Mosley - Personal Portfolio

This is my personal portfolio website that showcases my skills, projects, and contact information.

## How to View Locally

To view this website locally, follow these simple steps:

1.  **Navigate to the project directory:**
    Open your file explorer or terminal and go to the `C:\Users\ckens\Downloads\java\portfolio` directory.

2.  **Open `index.html`:**
    Locate the `index.html` file within the `portfolio` directory and open it with your preferred web browser (e.g., Chrome, Firefox, Edge, Safari).

That's it! The website should now be displayed in your browser.
